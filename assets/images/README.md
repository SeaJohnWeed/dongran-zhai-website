# 图片资源说明

## 个人照片

请从以下Google Scholar页面下载Dongran Zhai的个人照片：
- URL: https://scholar.google.com/citations?user=2O8Mr0oAAAAJ
- 保存为: `profile.jpg`
- 建议尺寸: 至少 400x400 像素
- 格式: JPG 或 PNG

## 使用方法

1. 访问上述Google Scholar链接
2. 右键点击个人照片
3. 选择"另存为"
4. 将图片保存到此目录，命名为 `profile.jpg`

## 注意事项

- 如果图片未加载，网站会尝试从Google Scholar直接加载
- 建议使用高质量图片以确保最佳显示效果
- 图片应为正方形或接近正方形比例
